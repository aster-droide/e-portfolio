Numerical Analysis

This module introduces key aspects of mathematics and statistics for data science and artificial intelligence. Each unit contains a Data Activity task with the results listed in this section. Aside from a first introduction in R, it is expected to gain a solid understanding of the NumPy library in Python.

Each unit below contains a short description of the learning objective, along with its respective Data Activity.
