Unit 2

DATA ACTIVITY UNIT 2

Crime Survey for England and Wales, 2013-2014: Unrestricted Access Teaching Dataset

This unit continues to work with the dataset introduced in unit 1.

For the data activity in this unit I'm tasked to explore whether survey respondents experienced any crime in the 12 months prior to the survey using the variable bcsvictim. I'm further tasked to create a frequency table, and assess whether this variable should be converted to a factor variable.

Upon first exploration I notice that this variable consists of binary values. My summary method from last week won't be very descriptive.

I explore the variable by creating a frequency table, where 0 = Not a victim of crime and 1 = victim of crime: 


# frequency table
> table <- table(ds[['bcsvictim']])
> table
0    1 
7460 1383 

# proportion table
> prop.table(table)
0         1 
0.8436051 0.1563949
So let's say roughly 16% of the sample population was a victim of crime in the 12 months preceding this survey.

I'm wondering whether the sex (1 = Male, 2 = Female) is relevant at all in the experience of crime: 


# cross classification frequency table
> table2 = table <- table(ds[['sex']], ds[['bcsvictim']])
> table2

0       1
1 3385  652
2 4075  731

# cross classification proportion table
> prop.table(table2)

0            1
1 0.38278865 0.07373063
2 0.46081647 0.08266425

# by margin
> prop.table(table2, margin = 1)

0           1
1 0.8384939 0.1615061
2 0.8478985 0.1521015
Men and women in this sample seem to experience the frequency of crime relatively similar.

When converting the variable to a factor, I'm not sure I see the benefit of this. The variable is already a binary representation as is, and presents more information on its internal structure than when this variable is converted to a factor:
